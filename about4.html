HTML window sub-frame

 Window sub-frame is a browser to display multiple windows, display multiple pages.


Window sub-frame

1.Role: The browser is divided into multiple windows, each window displays a different page (URL)

2.The role of various attributes:

<frame noresize="noresize" scrolling="null" src="http://www.baidu.com"> <!--noresize="noresize": Window can not be pulled, fixed frame size. scrolling="null":Hide the scroll bar-->


3.Code example

 	<!-- Definition: The browser is divided into multiple windows at the same time, each frame (page) has its own independent URL -->
 	<frameset rows="*,*,*">
 		<frameset cols="*,*">
 			<frame src="http://www.csdn.net">
 			<frame src="http://www.baidu.com">
 		</frameset>
 		
 		<frameset rows="*,*,*">
 			<frame src="http://www.sina.com">
 			<frame src="http://www.hao123.com">
 			<frame noresize="noresize" src="http://www.chinaunix.net"> <!-- noresize="noresize":  Window can not be pulled, fixed frame size -->
 		</frameset>
 		<frame scrolling="null" src="main.html"> <!-- scrolling="null":frameset rows -->
 	</frameset>
