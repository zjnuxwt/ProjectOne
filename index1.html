introduction



    Hypertext Markup Language, an application of the standard universal markup language. "Hypertext" means that the page can contain pictures, links, and even non-text elements such as music and programs. The structure of the Hypertext Markup Language includes a "header" part and a "body" part, where the "header" part provides information about the web page, and the "body" part provides the details of the web page.

  origin

  A hypermedia document on the World Wide Web calls it a page. A page where an organization or an   individual places a starting point on the World Wide Web is called a home page or home page, and   the home page usually includes a hyperlink to get a new web page by activation. The essence of the web page is the super-text markup language, by using a combination of other Web technologies, you can create powerful web pages. Thus, Hypertext Markup Language is the foundation of World Wide Web programming.

  definition

  The hypertext markup language marks each part of the web page to be displayed by a markup   symbol. Web page file itself is a text file, by adding a marker in the text file, you can tell the browser   how to display the contents. The browser reads the web page files in sequence and then interprets and displays the contents of their tags based on the tags.
